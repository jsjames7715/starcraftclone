# Contributing to StarCraft Clone

## Getting Started

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/your-username/starcraftclone.git
   ````
3. Install dependencies:
   ```bash
   npm install
   ```

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). Please read and adhere to it.

## Pull Request Process
1. Create a new branch
2. Make your changes
3. Commit and push
4. Open a pull request with a clear description of your changes
